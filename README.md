# MERN Stack Basic API Example

This example demonstrates a basic Express.js backend API, a core component of the MERN stack. It showcases handling GET, POST, PUT, and DELETE requests for a simple in-memory data store, illustrating fundamental backend concepts.

## Language

`javascript`

## How to Run

1. Save the code as `app.js`. 
2. Install dependencies: `npm install express body-parser`. 
3. Run the server: `node app.js`.
